# PASSGEN - Gerenciador de Senhas (Terminal)

Passgen é um sistema (CLI) gerador e gerenciador de senhas. Ele é um projeto da Unidade 2 da disciplina Linguagem de Programação II. Sua construção é inspirada nos sistemas web LastPass e Bitwarden que possuem o mesmo objetivo.

## Compilação e Execução
Para melhor experiência e aproveitamento visual do sistema, recomenda-se que o projeto seja compilado e executado diretamente no Terminal. Para isso, execute os comandos abaixo:

```
javac src/*.java src/**/*.java -d bin
java -cp bin Main
```
